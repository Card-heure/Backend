# Bienvenue sue le backend de Card'heure !
___
## Installation

- `git clone git@github.com:Card-heure/Backend.git`,
- `cd Backend`,
- `npm install`,
- `docker-compose up -d`,
- `cp .env.example .env`
- `node ace migration:run`,
- `node ace serve`
